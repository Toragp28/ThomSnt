Raccourcis Windows

  • Ctrl + C : Copier - Copie le contenu sélectionné.
  • Ctrl + X : Couper - Coupe le contenu sélectionné.
  • Ctrl + V : Coller - Colle le contenu copié/coupé.
  • Ctrl + Z : Annuler - Annule la dernière action.
  • Ctrl + Y : Rétablir - Rétablit la dernière action annulée.
  • Alt + Tab : Basculer entre les applications ouvertes.
  • Windows + D : Afficher le bureau.
  • Windows + E : Ouvrir l'Explorateur de fichiers.
  • Alt + F4 : Fermer la fenêtre active.
  • Ctrl + Shift + Esc : Ouvrir le Gestionnaire des tâches.
  • Windows + L : Verrouiller l'ordinateur.
  • Alt + F : Afficher le menu Fichier dans certaines applications.
  • Ctrl + Shift + N : Créer un nouveau dossier.
  • Windows + R : Ouvrir la boîte de dialogue Exécuter.
  • Alt + Entrée : Afficher les propriétés de l'élément sélectionné.
  • Ctrl + Shift + F : Rechercher des ordinateurs sur un réseau.
  • Alt + Flèche gauche/droite : Aller en arrière/avant dans certaines applications.
  • Ctrl + P : Imprimer le document actif.
  • Alt + Espace : Afficher le menu contextuel de la fenêtre active.
  • Shift + Suppr : Supprimer définitivement l'élément sélectionné sans le placer dans la Corbeille.
  • Alt + Maj : Basculer entre les langues de saisie (si plusieurs langues sont installées).
  • Ctrl + A : Sélectionner tout le contenu dans une fenêtre ou un champ de texte.
  • Alt + Impr écran : Capturer uniquement la fenêtre active.
  • Alt + F5 : Actualiser la fenêtre active.
  • Ctrl + Alt + Suppr : Afficher l'écran de sécurité de Windows (pour verrouiller l'ordinateur, changer d'utilisateur, etc.).
  • Alt + Maj + Tab : Basculer entre les applications ouvertes dans l'ordre inverse.
  • Ctrl + Esc : Ouvrir le menu Démarrer.
  • Alt + E : Ouvrir le menu Édition dans certaines applications.
  • Ctrl + D : Supprimer l'élément sélectionné et le placer dans la Corbeille.
  • Windows + Pause : Ouvrir les propriétés système.
  • Ctrl + N : Ouvrir une nouvelle fenêtre dans l'explorateur de fichiers.
  • Ctrl + S : Enregistrer le document actif.
  • Ctrl + O : Ouvrir un fichier.
  • Ctrl + F : Rechercher du texte dans le document actif.
  • Ctrl + H : Remplacer du texte dans le document actif.
  • Ctrl + E : Sélectionner la barre de recherche dans l'explorateur de fichiers.
  • Alt + Espace + N : Minimiser la fenêtre active.
  • Alt + Espace + X : Maximiser la fenêtre active.
  • Alt + Espace + R : Restaurer la fenêtre active.
  • Ctrl + Flèche gauche/droite : Déplacer le curseur d'un mot à gauche/droite.
  • Ctrl + Flèche haut/bas : Déplacer le curseur d'une ligne vers le haut/bas.
  • Shift + Flèche gauche/droite : Sélectionner du texte lettre par lettre.
  • Shift + Flèche haut/bas : Sélectionner du texte ligne par ligne.
  • Ctrl + Shift + Flèche gauche/droite : Sélectionner du texte mot par mot vers la gauche/droite.
